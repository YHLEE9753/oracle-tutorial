# 1. UNION
## syntax
```oracle-sql
SELECT
    column_list_1
FROM
    T1
UNION 
SELECT
    column_list_1
FROM
    T2;
```

```oracle-sql
SELECT
    column_list
FROM
    T1
UNION ALL 
SELECT
    column_list
FROM
    T2;
```
