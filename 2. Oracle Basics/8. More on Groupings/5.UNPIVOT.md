# 5. UNPIVOT
## syntax
- unpivot 를 사용하면 열과 행을 바꿀 수 잇다.
- 열과 행을 바꾸는 과정에서 데이터를 de-aggregate 하지 않는다는 점을 제외하면 unpivot 과 pivot 은 반대된다.

![img.png](../../images/unpivot.png)

```oracle-sql
SELECT 
    select_list
FROM table_name
UNPIVOT [INCLUDE | EXCLUDE NULLS](
    unpivot_clause
    unpivot_for_clause
    unpivot_in_clause 
);
```
